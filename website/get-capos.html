Deployment targets

Great for low-power NAS nodes, network appliances, and always-on home edge services.

x86 mini PCs

Perfect for file servers, media platforms, and automation workloads with stronger compute needs.

Virtual machines

Best for evaluation, CI testing, and staging before deployment to physical hardware.

Installation journey

01 · Download

Choose the proper image by architecture and device profile from releases.

02 · Flash

Write image to disk (SD card, SSD, or virtual disk) and verify checksum.

03 · Bootstrap

Boot the node, discover IP, set root password, and enable secure remote access.

04 · Configure

Set timezone, network policies, storage mounts, and required base services.

05 · Deploy apps

Install service stack for files, media, observability, or automation workloads.

Common post-install recommendations

Security baseline

Disable unnecessary services, apply least-privilege principles, and rotate critical credentials periodically.

Data safety

Adopt a 3-2-1 backup strategy, test restore workflows, and monitor disk health continuously.

Operational readiness

Set update windows, define rollback plans, and keep lightweight monitoring enabled by default.
