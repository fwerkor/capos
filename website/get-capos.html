GET CAPOS

下载、刷写、启动你的 CapOS 节点

你可以直接使用官方编译版本，也可以根据硬件架构自行构建镜像。建议先在测试环境验证，再迁移到正式 NAS 或边缘节点。

GitHub Releases FWERKOR Repo

部署方式

路由器 / ARM 设备

适用于低功耗 NAS、网关与家庭边缘服务。充分利用轻量设计与高可用网络能力。

x86 小主机 / 旧电脑

可构建个人数据中心，运行文件管理、媒体服务与自动化任务，兼顾性能与成本。

虚拟机环境

推荐用于学习、测试与 CI 验证，便于快照回滚和多版本对比。

默认访问端口（README 同步）

Web Panelhttp: 2000/tcp · https: 2020/tcp
SSH22/tcp（设置 root 密码后）
Telnet23/tcp（仅初始阶段）
网络默认DHCP + DHCPv6

快速验证

# 首次登录后建议立即修改凭据
passwd

# 查看服务状态
service status

# 按需安装扩展包（示意）
opkg update
opkg list | head -n 20
