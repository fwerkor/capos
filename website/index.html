OPEN SOURCE · NAS · EDGE SERVER

Build your next home lab on CapOS

CapOS is an open-source NAS and server operating system designed for home labs, edge devices, and lightweight infrastructure. It combines a clean WebDesktop experience with a package-rich Linux ecosystem, so both beginners and advanced users can deploy services quickly.

WHY CAPOS

Everything you need for a modern open NAS platform

Lightweight by Design

Small footprint and efficient resource usage, ideal for routers, mini PCs, virtual machines, and repurposed hardware.

Powerful Ecosystem

Built on a mature Linux package ecosystem so you can install services fast and keep your stack flexible over time.

WebDesktop + CLI

Use the visual web interface for day-to-day management and switch to CLI when you need fine-grained control.

Extensible Architecture

Add apps and services as your needs grow—from file sharing and media servers to automation and edge workloads.

Secure Baseline

Sensible defaults and stable update practices help keep your home lab and server environment reliable.

Community Driven

GPL-licensed and open for contributions through issues, discussions, and pull requests on GitHub.

USE CASES

Run real workloads, not just demos

家庭数据中心

将家庭照片、视频、文档和备份集中到一台设备，通过 WebDesktop 完成权限配置、共享发布和基础运维。

创客与边缘实验室

在低功耗设备上运行 MQTT、自动化脚本、轻量数据库和监控组件，构建可扩展的本地边缘服务。

小团队服务节点

用于内部文件协作、镜像缓存、CI 辅助节点或测试环境，快速部署并保持可迁移性与可维护性。
