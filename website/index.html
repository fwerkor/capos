OPEN SOURCE · NAS · EDGE SERVER

把 OpenWrt 的稳定内核 带到现代 NAS 与家庭实验室体验里

CapOS 本质是一个开源 NAS / 轻量服务器操作系统：继承 OpenWrt 生态优势，强调轻量与可扩展；同时提供 WebDesktop 管理体验，让非专业用户也能完成服务部署、系统运维与资源管理。

WHY CAPOS

围绕“生态、扩展、轻量、易用”打造

生态强大

CapOS 基于 OpenWrt 技术体系，天然具备成熟包生态与多架构兼容基础，既适合老硬件再利用，也利于持续演进。

扩展性强

你可以按需叠加服务与模块，构建个人 NAS、下载中心、家庭自动化中枢，或作为边缘节点运行专业应用。

轻量而实用

延续 OpenWrt 的小体积与低资源占用特性，在路由器、x86 主机、虚拟机等环境都能平稳运行。

WebDesktop 管理

通过浏览器即可完成配置、监控和软件安装；相比纯命令行方案，对新手更友好，对老手更高效。

安全与稳定

遵循稳健系统基线，默认防火墙策略清晰，并持续跟进内核与软件包更新，兼顾长期可维护性。

真正开源

GPL-2.0 许可，欢迎社区通过 Issue、Discussion、PR 参与建设，让 CapOS 持续变得更完整。
