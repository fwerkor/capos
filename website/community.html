COMMUNITY

把 CapOS 做成更好的 开源 NAS 系统

CapOS 欢迎任何形式的贡献：问题反馈、功能建议、代码优化、文档改进、应用生态建设。只要你愿意参与，就能让项目更快进化。

参与方式

Issues

提交 bug、兼容性问题和需求建议，帮助维护者快速定位优先级。

打开 Issues

Discussions

发起方案讨论、部署经验交流和新特性 brainstorm，沉淀社区共识。

进入 Discussions

Pull Requests

提交代码与文档改进，建议附带可复现步骤和测试结果，便于评审合并。

查看 PR

我们最期待的贡献

  • 不同硬件架构的镜像测试与适配反馈。
  • WebDesktop 交互优化与可用性改进。
  • 应用模板、部署脚本和最佳实践文档。
  • 安全策略审查与性能压测数据。

协作建议

建议在提交 PR 前先开 Issue 讨论方向，减少重复开发；同时保持提交颗粒度清晰，便于 review 与回滚。
